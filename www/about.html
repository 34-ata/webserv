🚀 WebServ HTTP Server

A professional HTTP/1.1 server implementation in C++98 with modern web capabilities.

✨ Core Features

📄 HTTP Methods

  • GET: File retrieval and content serving
  • POST: File creation and data processing
  • DELETE: File removal operations

⚙️ CGI Processing

  • Python script execution
  • Environment variable handling
  • Dynamic content generation

📁 File Management

  • Static file serving
  • Directory indexing
  • MIME type detection
  • File upload handling

�️ Error Handling

  • Custom error pages
  • Graceful failure handling
  • Request validation

🔬 Technical Specifications

Language:C++98
HTTP Version:HTTP/1.1
I/O Model:Non-blocking poll()
CGI Standard:CGI/1.1
Configuration:Nginx-style syntax
